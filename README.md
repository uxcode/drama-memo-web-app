
## 메모 웹 애플리케이션

### 실행환경

도커가 설치 되어 있어야 합니다.
https://hub.docker.com/editions/community/docker-ce-desktop-mac


### 실행방법

프로젝트 루트 폴더에서 아래 스크립트 실행

##### 설치

`./bin/setup.sh`

##### 시작

`./bin/run.sh`

##### 중지

`./bin/stop.sh`

##### 삭제

`./bin/destroy.sh`

##### 앱 주소

`http://localhost:8000`


### 프로젝트 구성

React, Bootstrap, Docker
